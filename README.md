# IT Company Website

## Project Description
This project is a simple website for an IT company. It includes several pages that provide information about the company, its services, projects, and contact details. The website is structured to be user-friendly and informative.

## Project Structure
```
it-company-website
├── index.html
├── about.html
├── services.html
├── projects.html
├── contact.html
├── css
│   └── style.css
├── js
│   └── script.js
└── README.md
```

## File Descriptions
- **index.html**: The main entry point for the website, linking to other pages and resources.
- **about.html**: Contains information about the company, including its mission, vision, and team.
- **services.html**: Details the services offered by the company, highlighting features and benefits.
- **projects.html**: Showcases completed projects with descriptions and links to demos or case studies.
- **contact.html**: Provides contact information and a form for inquiries.
- **css/style.css**: Contains styles for the website, defining layout, colors, and fonts.
- **js/script.js**: Includes JavaScript for interactivity, such as form validation and animations.

## Setup Instructions
1. Clone the repository or download the project files.
2. Open the project in your preferred web browser to view the website.
3. Modify the HTML, CSS, and JavaScript files as needed to customize the website for your company.
